A modern, interactive metrics dashboard built with React Native and Expo. This application features a bottom tab navigation system that displays four different types of charts - Line, Bar, Pie, and Progress charts - each in its own dedicated screen for optimal viewing experience.

## âœ¨ Features

- **Bottom Tab Navigation**: Intuitive 4-tab interface for seamless chart browsing
- **Line Chart**: Daily app usage metrics with smooth curves and gradient fill
- **Bar Chart**: Weekly usage statistics with color-coded visualization
- **Pie Chart**: Category breakdown with dynamic color generation
- **Progress Chart**: Feature completion tracking with circular indicators
- **Real-time Data**: Dynamic API integration using axios
- **Responsive Design**: Adapts to different screen sizes automatically
